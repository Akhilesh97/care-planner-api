# API Examples

Use these examples locally or after deploying to Render.

Local base URL:

```text
http://localhost:8001
```

Render base URL:

```text
https://<your-render-service>.onrender.com
```

Replace `<your-render-service>` with your actual Render service name.

## `GET /health`

Checks that the API is running and shows the configured LLM provider.

Local:

```bash
curl http://localhost:8001/health
```

Render:

```bash
curl https://<your-render-service>.onrender.com/health
```

Example response:

```json
{
  "status": "ok",
  "service": "Neev Care Planner API",
  "llm_provider": "gemini",
  "model": "gemini-2.5-flash-lite"
}
```

## `GET /docs`

Opens the interactive Swagger API docs.

Local:

```text
http://localhost:8001/docs
```

Render:

```text
https://<your-render-service>.onrender.com/docs
```

## `POST /care-plan`

Generates a 7-day care plan. Day 1 activities are supplied by the caller. Days 2 through 7 are generated by the configured LLM.

### Example: motor development priority

Local:

```bash
curl -X POST "http://localhost:8001/care-plan" \
  -H "Content-Type: application/json" \
  -d '{
    "mode": "MONITOR",
    "age_months": 2,
    "domain1": {
      "name": "Motor Development",
      "zscore": 63.5
    },
    "domain2": {
      "name": "Communication",
      "zscore": 72.0
    },
    "domain3": {
      "name": "Social Engagement",
      "zscore": 70.5
    },
    "day1_activities": [
      {
        "title": "Tummy Time Reach",
        "description": "Place baby on tummy and hold a bright toy just out of reach.",
        "image_url": null
      },
      {
        "title": "Supported Sitting Wobble",
        "description": "Sit behind baby and gently tilt them side to side.",
        "image_url": null
      },
      {
        "title": "Kick and Bat",
        "description": "Hold a rattle above feet then hands and encourage kicking or batting.",
        "image_url": null
      }
    ]
  }'
```

## Common Error Examples

### Missing LLM API key

If `LLM_PROVIDER=gemini` but `GEMINI_API_KEY` is missing:

```json
{
  "detail": "GEMINI_API_KEY is not set in environment / .env file."
}
```

If `LLM_PROVIDER=openai` but `OPENAI_API_KEY` is missing:

```json
{
  "detail": "OPENAI_API_KEY is not set in environment / .env file."
}
```

### Invalid request body

If `day1_activities` does not contain exactly 3 items, FastAPI returns a `422 Unprocessable Entity` response with validation details.
